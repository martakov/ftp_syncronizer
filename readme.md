# PHP ftp syncronizer

Usage
_____

### Config

host: 00.00.00.0
user_name: user_name
password: user_password
ftp_directory: /wordpress
local_directory: absolute path to the directory
pid_file: ./storage/pid

### Run deamon from console

php ./bin/console.php -d start

### Stop deamon

php ./bin/console.php -d stop
